# Profile Management API

## Tech Stack
- Node.js
- Express.js
- MongoDB
- React (Frontend)

## Setup Instructions

1. Clone repository
git clone <repo-link>

2. Install dependencies
cd server
npm install

cd ../client
npm install

3. Run backend
cd server
npm run dev

4. Run frontend
cd client
npm start

## Innovation Feature
- Real-time profile update
- Skills endorsement system
- REST API structure
- Clean folder architecture
